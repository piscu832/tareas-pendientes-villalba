# TAREAS PENDIENTES - Kanban Board (Villalba)

Una aplicación de gestión de tareas tipo Kanban moderna, diseñada con la identidad visual de **VILLALBA**.

## 🚀 Características
- **Diseño Institucional**: Colores y tipografía alineados con la marca Villalba.
- **Tablero Kanban**: Columnas de Pendiente, En Proceso y Terminado.
- **Priorización Automática**: Las tareas más urgentes siempre aparecen arriba.
- **Base de Datos en la Nube**: Integración opcional con Supabase para persistencia multiplataforma.
- **PWA Ready**: Funciona localmente mediante LocalStorage como fallback.

## 🛠️ Tecnologías
- **Frontend**: HTML5, Vanilla JavaScript.
- **Estilos**: Tailwind CSS (CDN), CSS3 personalizado.
- **Base de Datos**: Supabase (PostgreSQL).

## 📦 Instalación y Despliegue
1. Clona este repositorio.
2. Abre `index.html` en tu navegador.
3. Para persistencia en la nube, configura tus credenciales de Supabase en `app.js`.
